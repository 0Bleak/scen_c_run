## tc Traffic Shaping — THE KEY FIX
tc on ogstun creates genuine DL scarcity (emulates FRMCS congested backhaul).
Must be applied AFTER UEs attach and IPs are known.
Must be reapplied every session (does not survive reboot).
Install if missing: sudo dnf install -y iproute-tc

CAPS: BUSINESS=2Mbit, CRITICAL=150kbit, PERFORMANCE=120kbit

TEMPLATE (replace IPs per session):
sudo tc qdisc del dev ogstun root 2>/dev/null; true
sudo tc qdisc add dev ogstun root handle 1: htb default 10 r2q 1
sudo tc class add dev ogstun parent 1: classid 1:10 htb rate 1000mbit burst 15k quantum 1514
sudo tc class add dev ogstun parent 1: classid 1:1  htb rate 2mbit   burst 1500b cburst 1500b quantum 1514
sudo tc class add dev ogstun parent 1: classid 1:2  htb rate 150kbit burst 1500b cburst 1500b quantum 1514
sudo tc class add dev ogstun parent 1: classid 1:3  htb rate 120kbit burst 1500b cburst 1500b quantum 1514
sudo tc qdisc add dev ogstun parent 1:1 handle 10: tbf rate 2mbit   burst 1500b latency 1ms mtu 1500
sudo tc qdisc add dev ogstun parent 1:2 handle 20: tbf rate 150kbit burst 1500b latency 1ms mtu 1500
sudo tc qdisc add dev ogstun parent 1:3 handle 30: tbf rate 120kbit burst 1500b latency 1ms mtu 1500
sudo tc filter add dev ogstun parent 1: protocol ip u32 match ip dst <BUSINESS_IP>/32  flowid 1:1
sudo tc filter add dev ogstun parent 1: protocol ip u32 match ip dst <CRITICAL_IP>/32  flowid 1:2
sudo tc filter add dev ogstun parent 1: protocol ip u32 match ip dst <PERF_IP>/32      flowid 1:3

VERIFY tc is working:
sudo tc -s class show dev ogstun | grep -A3 "class htb 1:[123] "
DU must show: BUSINESS ~1-2M, CRITICAL ~120-200k, PERFORMANCE ~100-170k

## MDP (identical across PPO and DQN)
State (6): [sat_C, sat_P, sat_B, cqi_C/15, cqi_P/15, cqi_B/15]
  sat_i = dl_brate_i / SLA_DL_i clipped to 1.5
Action (6 discrete PRB profiles [C,P,B] sum=100):
  0:[34,33,33] 1:[60,25,15] 2:[25,60,15] 3:[15,25,60] 4:[45,45,10] 5:[10,10,80]
Reward: 0.6*min(sat_C,1) + 0.3*min(sat_P,1) + 0.1*min(sat_B,1) - 0.3*[sat_C<1]
Mask: forbid profiles with CRITICAL<30 when CRITICAL unsatisfied
SLA_DL: CRITICAL=356k, PERFORMANCE=311k, BUSINESS=20M
